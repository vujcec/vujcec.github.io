Local weather

Current local conditions are shown below using wttr.in in browser mode.

This page redirects automatically to the GitHub profile after 2 seconds. The weather is embedded using the browser-oriented view to avoid separate text files opening.

Weather is location-based by IP address, so the result can be approximate on VPNs, mobile networks, or some ISPs. If you still want the one-line text format specifically, it is better used in terminal bars or through a proxy/self-hosted bridge rather than directly in a browser page.
